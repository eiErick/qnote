## Installation

Install qnote with snap

```bash
  snap install qnote
  snap run qnote
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/eierick/qnote
```

Go to the project directory

```bash
  cd qnote
```

Install dependencies

```bash
  npm install
```

Start the qnote

```bash
  npm run start
```
